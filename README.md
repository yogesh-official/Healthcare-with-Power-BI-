# Healthcare Power BI Dashboard

This project contains a **Healthcare Dashboard** built using Power BI, providing key insights and visualizations for hospital management. The dashboard includes patient summaries, hospital performance metrics, and various visual components for analysis of healthcare data. It's designed for use by healthcare administrators, managers, and analysts to enhance decision-making processes.

## Features

* **Inpatient/Outpatient Data:** Displays the number of inpatient and outpatient patients, helping to monitor hospital capacity.
* **Weekly Trends:** Shows the trend of patient in/out statistics over time.
* **Average Treatment Cost:** Provides insight into the cost of treatments for patients.
* **Revenue Insights:** Breaks down total revenue by age group.
* **Patient Feedback:** Tracks how well doctors explain treatment to patients.
* **Departmental Performance:** Highlights ER times and other department-specific data.

### Key Visualizations

* **Total Patient Overview**: Breakdown of total, inpatient, and outpatient patients.
* **Patient Statistics**: Details of patients by age, gender, and their feedback on treatment.
* **Bed Occupancy**: Real-time display of bed occupancy in the hospital.
* **Revenue by Age Group**: A detailed breakdown of revenue generated across different age groups.
* **Emergency Room (ER) Time**: Average ER time by department and age group.

## Setup

1. **Download the Power BI File**
   Download the `.pbix` file from this repository to start working with the dashboard in Power BI.

2. **Prerequisites**

   * Install **Power BI Desktop** to view and modify the dashboard.
   * Ensure access to relevant healthcare data for further customizations.

3. **Customization**

   * You can add additional data sources or update the existing ones to match your hospital's real-time data.
   * Update parameters such as **Age Group**, **Patient Count**, or **Revenue** to reflect the most current metrics.
